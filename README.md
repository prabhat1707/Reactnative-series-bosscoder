# React Native Series

Welcome to the React Native Series project! This repository contains a collection of demonstrations and examples covering essential React Native concepts.

## What's in this project

This project is divided into several key areas, each focusing on a fundamental aspect of React Native development:

### 1. UI Demo
*Examples of building user interfaces.*
- Focuses on utilizing core React Native components (Views, Text, Images, etc.).
- Demonstrates styling using `StyleSheet`, layout management with Flexbox, and building responsive designs.

### 2. API Demo
*Interacting with external data sources.*
- Covers fetching and sending data to REST APIs.
- Includes handling asynchronous requests, managing loading states (spinners/skeletons), and handling network errors gracefully.

### 3. Navigation
*Managing screen transitions and application flow.*
- Implementation of routing using tools like **React Navigation**.
- Contains examples of different navigation patterns such as Stack Navigators, Tab Navigators, or Drawer Navigators.

### 4. Redux
*Global state management.*
- Demonstrates setting up a Redux store, defining actions, and writing reducers.
- Shows how to connect components to the Redux store to manage and access global application state across different screens.

## Getting Started

To run this project locally on your machine, ensure you have your React Native environment completely set up.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/prabhat1707/Reactnative-series-bosscoder.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start the Metro Bundler:**
   ```bash
   npm start
   # or
   yarn start
   ```

4. **Run on your preferred platform:**
   ```bash
   npm run android  # For Android emulator/device
   npm run ios      # For iOS simulator/device
   ```
